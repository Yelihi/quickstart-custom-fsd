## Context

### 1. FSD(Feature-Sliced Design)

해당 아키텍쳐에 대한 자세한 설명은 [FSD 공식문서](https://feature-sliced.design/docs/get-started/overview)를 참고한다.

### 2. 실제 프로젝트에 적용하기

공식문서 내 FSD 구조를 그대로 적용하기에는 한계점들이 존재하고, 이에 도메인 주도 설계(DDD)를 기반으로 변형된 FSD 구조를 사용한다.
자세한 설명은 [현 프로젝트 구조](reference/FSD.md) 설명과 규칙을 기반으로 구현할 수 있도록 한다.
